# Frontend

## Запуск проекта
``` bash
# установить зависимости, используется ярн2
$ yarn

# разработка с авто-обновлением - `http://127.0.0.1:5173/`
$ yarn dev

# продакшн-сборка и ее запуск
$ yarn build
$ yarn start

# генерация статичного проекта - нужен для деплоя, появится директория dist
$ yarn generate
```
